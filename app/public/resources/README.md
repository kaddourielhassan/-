# Ressources multimedia

Ce dossier contient les ressources reutilisables du projet web.

## Structure

- `images/` : logos, illustrations, icones, fonds
- `audio/` : effets sonores, musiques d'ambiance, voix

## Nommage recommande

- Utiliser des noms en minuscules avec tirets (ex: `menu-click.mp3`).
- Eviter les espaces et caracteres speciaux.
- Regrouper par type dans les sous-dossiers prevus.

## Formats conseilles

- Images: `png`, `jpg`, `webp`, `svg`
- Audio: `mp3`, `wav`, `ogg`

## Exemple d'usage dans le front

- Image: `/resources/images/logo.png`
- Audio: `/resources/audio/menu-click.mp3`

## Vocabulaire (app arabe)

- Les fichiers attendus pour les flashcards sont documentes dans `MEDIA_CHECKLIST.md`.
- Dossiers cibles:
  - `/resources/images/vocabulaire/`
  - `/resources/audio/vocabulaire/`
