## **GROUPE 2: 6 PAIRES PHONÈMES** (6 fichiers - 1 paire = 1 fichier)

### Comment Enregistrer
- **Chronologie** : Son 1, pause 2s, Son 2
- **Répétition** : 2-3x chaque son
- **Exemple** : "Haa (guttural)... Haa (glottal)... Haa... Ha"

### Détail des Paires Critiques

**Paire 1: ح (HAA guttural) vs ه (HA glottal)**
```
File: phoneme_01_haa_vs_ha.mp3
Contenu: [ح] guttural puis [ه] glottal
Astuce: ح = son profond dans gorge
        ه = expiration légère
```

**Paire 2: ع ('AYN) vs أ (ALIF)**
```
File: phoneme_02_ayn_vs_alif.mp3
Contenu: [ع] guttural puis [أ] coup de glotte
Astuce: ع = constriction gorge
        أ = son A normal
```

**Paire 3: ص (SAAD emphatique) vs س (SIIN simple)**
```
File: phoneme_03_saad_vs_siin.mp3
Contenu: [ص] (S emphatique) puis [س] (S normal)
Astuce: ص = bouche arrondie, son "grave"
        س = son S français
```

**Paire 4: ض (DAAD emphatique) vs د (DAAL simple)**
```
File: phoneme_04_daad_vs_daal.mp3
Contenu: [ض] puis [د]
Astuce: ض = D "heavy"
        د = D normal
```

**Paire 5: ط (TAA emphatique) vs ت (TA simple)**
```
File: phoneme_05_taa_vs_ta.mp3
Contenu: [ط] puis [ت]
Astuce: ط = T profond
        ت = T normal
```

**Paire 6: ق (QAF uvulaire) vs ك (KAAF velaire)**
```
File: phoneme_06_qaf_vs_kaaf.mp3
Contenu: [ق] uvulaire puis [ك] velaire
Astuce: ق = gorge arrière très profonde
        ك = K normal
```

**Total: 6 fichiers**

## **3️⃣ SOURCES & OUTILS RECOMMANDÉS**

### **Pour les AUDIOS:**

**Option 1: Vous enregistrez (MEILLEUR pour la maîtresse - authenticité)**
- Outil: Téléphone (dictaphone) OU GarageBand (Mac)
- Avantage: Voix authentique, enfants reconnaissent maîtresse
- Qualité: Suffisant pour MVP

**Option 2: Créer syntétiquement (Backup)**
- Google Translate (TTS arabe)
- Limitation: Accent parfois fake
- Utiliser si enregistrement maîtresse indisponible

### **Pour les IMAGES:**

**Option 1: Canva (FACILE)**
- Site: https://www.canva.com
- Avantage: Templates enfants, design pro, gratuit
- Rechercher: "Arabic letters", "Education", "Kids"
- Export: PNG transparent

**Option 2: Pixabay / Unsplash (Libre)**
- Télécharger images libres
- Avantage: Gratuit, qualité OK
- Limitation: Style moins cohérent

**Option 3: Combiner (Meilleur)**
- Canva pour design lettres + backgrounds
- Pixabay pour objets/animaux
- Résultat: Style cohérent + variété

## **⚠️ NOTES IMPORTANTES**

1. **Qualité Audio > Perfection**
   - Pas besoin studio professionnel
   - Voix maîtresse > voix synthétique
   - Juste éviter bruit ambiant (silence)

2. **Cohérence Visuelle**
   - Toutes images même style (1 designer)
   - Palette couleurs limitée (max 8-10 couleurs)
   - Polices: Max 2-3 fonts (1 arabic, 1 sans-serif)

3. **Accessibilité**
   - Audio: Volume constant (pas de pics)
   - Images: Contraste fort (lisible sur petit écran)
   - Emojis: Eviter si images dedans

4. **Timeline**
   - Idéal: Préparer S0 (avant S1)
   - Minimum: Jour 2 (pendant S1)
   - Critical: Audio avant Jour 2 midi (vs Bubble)
